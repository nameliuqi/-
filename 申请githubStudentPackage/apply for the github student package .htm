如何申请github学生包

 

扯淡先说我为啥要申请这个

最近搞了个chromebook，谷歌的东西嘛，肯定要翻墙的，于是就翻了之前翔大的科学上网教程，于发现了vps+ss的组合，并且在解奥然大神的指导下翻墙成功，于是想自己独立搞个大新闻，就开始折腾。

开始用的搬瓦工，2.99刀一个月用于实验。之前知道digitalocean在github学生包里有50刀的代金券（好久之前是100刀的），于是准备申请。

按照基本法来说应该是用edu邮箱验证个身份就好了，但一切按照国情来，我国edu邮箱持有者太多，于是国外的公司对持有者学生身份保持高度怀疑，在申请了几次之后被秒rejected。

于是开始百度（个人认为百度还是可以用用的，中文支持不错，贴近国情，但是内容上还是谷歌好一点），并且给他们发邮件，发现知乎大牛说可以直接申请，于是再次上网站。

干货过程

在申请的时候填写没有学生邮箱(I don’t have student email)，之后才会让你上传凭证，这时把自己的学生证，教务处网站截图上传，并且用ps什么的标注好自己的入学年份和毕业年份。之后就OK了。一段时间之后，一般是一周，就会发现approved。然后在网站上通过提示的方式（有的是链接，有的是优惠码）就可以获得相应的优惠。

备注：

1. edu邮箱还是有用的，例如比较好的ide，微软的dreamspark都可以用（但是微软的我只是通过了验证，还没开始使用，大家自己探索）

2. 如果闲麻烦可以购买搬瓦工，20刀一年，可以一键搭建openvpn和ss

3. 水平有限，如果有错误请指正
