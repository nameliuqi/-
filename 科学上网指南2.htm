digitalocean方案

前言：

5刀一个月，不支持支付宝，支持paypal（paypal可以用银联的卡支付，如果不想麻烦可以联系我）。

但是githubstudentpackge（更多的用途见链接）有50刀的代金券，注册送10刀，通过邀请链接10刀，

基本够用一年（可以一个人申请，多人合用，到期了然后另外一个人申请）。

流程：

 

1. 申请github学生包

  1. 注册github账号（废话），找到学生包链接
  2. 申请，一堆填理由（申请理由见附录，别照抄，说不定查重。。）信息，然后选择i dont have edu××（我国edu邮箱持有者太多，于是国外的公司对持有者学生身份保持高度怀疑，工大edu邮箱在github学生包申请上上已经不能用了），之后会让你上传凭证（这里详细说一下，如果你用学生邮箱验证的话，github是通过学生邮箱判断你的学生身份，正常情况下是被秒拒绝的，但是如果选择你没有学生邮箱的话你就可以上传附件，从而通过人工审核判断你的学生身份）
  3. 上传凭证，这时把自己的学生证照片，教务处网站截图上传，并且用ps什么的用英语标注好自己的入学年份和毕业年份。
  4. 等待。之后就OK了。一段时间之后，一般是一周，就会发现approved。
  5. 其他：如果出现问题，发邮件交流。详细邮件内容见后文附录。edu邮箱还是有用的，例如比较好的ide（pycharm，phpstrom），微软的dreamspark都可以用（但是微软的我只是通过了验证，还没开始使用，大家自己探索）。

2. 注册digitalocean

正常的注册流程 。注意自己的邮箱别错。

安利一下我的优惠链接。你我都可以获得10刀。

3. 充值，你才能变得更强。

注册之后并不能马上使用，需要充值5刀表示这是一个正常的账户。充值方式可以是信用卡，也可以是palpay。如果你有visa卡就随便充值了。没有的话paypal可以支持部分银联卡，另外，如果搞不定可以联系我让我帮你代付。充值之后在support里面找客服解锁账户（使账户正常使用）。如果有问题，digitalocean的support十分好，开一个ticket然后问问题，很快就得到了答复。

4.购买一个droplet

droplet我理解就是你的一个主机。先用你的优惠码。如果你现购买droplet之后你就不是一个新用户，就不能用优惠码了。这时候你如果还想要这50刀就去找客服吧。之后购买droplet，机房选新加坡或者旧金山的。配置ss最便宜的能够支持5人以上使用。

补充1：

优惠码不能使用时,通过support找到客服,说明情况,大体英文如下

Hello , I am a student and I have a 50$ code that I get from github student package , but it seems like I can’t use it now . I need your help. This is my code  “你从github获得的代码”.

补充2:

另外的，如果你不只想用vps搭建ss，还想搞个人网站，在搭建的时候可以有一键搭建应用，例如lamp和wordpress。

5.搭建ss

教程链接

教程中配置说明（from wiki）：

  {

    "server": "0.0.0.0",//监听的端口

    "server_port": 5173,//你计算机上配置的端口

    "local_port": 1080,//vps上的端口

    "password": "your_password",//密码

    "timeout": 600,//超时

    "method": "aes-256-cfb"//加密方式

  }

 

vim（在搞这个东西的时候我遇到的问题之一就是不会用vim，可以自行百度）：

    # vim /etc/shadowsocks.json

    进入 vi / vim 的编辑状态后，按 I 键或者 Insert 键切换到 Insert 模式，
    编辑完成以后，先按 Esc 键退出 Insert 模式，再输入 :wq (冒号wq)，并回车，意思是保存并且退出，即可。
